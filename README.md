# Employee Management System

## 🚀 Tech Stack
- Backend: Spring Boot
- Frontend: React + TypeScript
- Database: MySQL

## 📌 Features
- Add, Update, Delete Employees
- REST APIs
- Responsive UI

## ▶️ Run Project

### Backend
cd backend
mvn spring-boot:run

### Frontend
cd frontend
npm install
npm start
